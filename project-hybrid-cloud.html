Cloud Security Case Study

Hybrid Cloud Monitoring and Resilience Lab

A hybrid cloud environment across AWS, Azure, IBM Cloud, and VMware with centralized logging, automation, IAM controls, and resilience-focused monitoring.

Project Overview

This project demonstrates my ability to design and work with multi-cloud and hybrid infrastructure. The lab focused on cloud monitoring, automated provisioning, centralized log collection, IAM controls, and processing large volumes of security telemetry.

What I Worked On

  • Built a hybrid cloud environment across AWS, Azure, IBM Cloud, and VMware.
  • Configured centralized logging and basic IAM controls to support monitoring and access management.
  • Automated VM provisioning and monitoring workflows using Terraform and Python.
  • Used ELK Stack to support infrastructure monitoring and log visibility.
  • Designed big data pipelines using Apache Spark and Apache Kafka for security telemetry and log data.
  • Implemented Python scripts for data transformation and security log processing across cloud systems.

Project Outcomes

  • Strengthened practical knowledge of hybrid cloud architecture and infrastructure monitoring.
  • Developed automation experience using Python and Terraform.
  • Connected cloud monitoring with cybersecurity log processing and resilience planning.
