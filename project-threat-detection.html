Threat Detection Case Study

Security Log Analysis and Threat Detection

A security analytics pipeline using Splunk, ELK Stack, Python, Scikit-learn, and PyTorch to analyze endpoint and network security logs.

Project Overview

This project focused on security monitoring, log analysis, anomaly detection, and incident response support. It demonstrates my interest in using data and AI techniques to improve security investigations and threat detection.

What I Worked On

  • Built a security analytics pipeline using Splunk, ELK Stack, and Python.
  • Analyzed large-scale endpoint and network security logs.
  • Applied machine learning techniques with Scikit-learn and PyTorch.
  • Focused on detecting unauthorized access, brute-force attacks, and anomalous login patterns.
  • Created a standardized incident response playbook.
  • Used AI-generated insights to support faster security investigations.

Project Outcomes

  • Built stronger understanding of security analytics and threat detection workflows.
  • Connected machine learning concepts to real-world security problems.
  • Created a repeatable approach for incident response documentation and investigation.
